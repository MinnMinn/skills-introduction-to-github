# Contributing

Thank you for your interest in contributing to this project! We welcome contributions of all kinds.

## How to Run Tests

Follow the steps below to run the test suite locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/MinnMinn/skills-introduction-to-github.git
   cd skills-introduction-to-github
   ```

2. **Install dependencies**

   Install any required dependencies before running tests. For example, if the project uses Node.js:

   ```bash
   npm install
   ```

   Or, for a Python project:

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the tests**

   Execute the full test suite with:

   ```bash
   # Node.js / JavaScript
   npm test

   # Python
   pytest
   ```

4. **Check the results**

   All tests should pass before submitting a pull request. If any tests fail, please review the output and fix the issues before opening your PR.

---

If you have any questions, feel free to open an issue or start a discussion. Happy contributing! 🎉
